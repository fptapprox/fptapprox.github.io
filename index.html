List of FPT+Approximation results

Please send any comments/corrections/updates to fptapprox@gmail.com


  • Strongly Connected Steiner Subgraph (SCSS): Given a digraph \(G=(V,E)\) and a set of terminals \(T=\{t_1, t_2, \ldots, t_k\}\) is there a set \(H\subseteq V\) of size \(\leq p\) such that \(G[H]\) has an \(t_i \leadsto t_j\) path for each \(1\leq i\neq j\leq k\)
    Algorithm: 2-approximation in \(2^{k}\cdot n^{O(1)}\) time [N13]
    Lower Bound: No \((2-\epsilon)\)-approximation in \(f(k)\cdot n^{O(1)}\) time (for any function \(f\)) under Gap-ETH [CFM18]

  • Directed Odd Cycle Transversal (DOCT): Given a digraph \(G=(V,E)\) is there a set \(H\subseteq V\) of size \(\leq k\) such that \(G[V\setminus H]\) has no (directed) odd cycles
    Algorithm: 2-approximation in \(2^{O(k^2)}\cdot n^{O(1)}\) time [LRSZ17]
    Lower Bound: There is a constant \(\epsilon>0\) such that there is no \((1+\epsilon)\)-approximation in \(f(k)\cdot n^{O(1)}\) time (for any function \(f\)) under Parameterized Inapproximability Hypothesis (PIH) and FPT\(\neq\) W[1] [LRSZ17]

  • Directed Steiner Network (DSN): Given a digraph \(G=(V,E)\) and a set of terminal pairs \(T=\{(s_i, t_i)\ :\ i\in [k]\}\) is there a set \(H\subseteq V\) of size \(\leq p\) such that \(G[H]\) has an \(s_i \leadsto t_i\) path for each \(i\in [k]\)
    Algorithm: \(k^{1/2+\epsilon}\)-approximation in polynomial time for every \(\epsilon>0\) time [CEGS11, FKN12]
    Lower Bound: No \((k^{1/4-o(1)})\)-approximation in \(f(k)\cdot n^{O(1)}\) time (for any function \(f\)) under Gap-ETH [DM18]
    Special Case 1: When the input graph is bidirected, i.e., for every edge \(u\rightarrow v\) the reverse edge \(v\rightarrow u\) is also present and moreover has same weight, then the problem is called as BI-DSN. For this problem, there is an \(2\)-approximation in \(2^{k\log k}\cdot n^{O(1)}\) time and on the lower bound side it is known there is a constant \(\epsilon>0\) such that there is no \((1+\epsilon)\)-approximation in \(f(k)\cdot n^{O(1)}\) time (for any function \(f\)) under Gap-ETH [CFM18]
    . Special Case 2: When the input graph is bidirected, i.e., for every edge \(u\rightarrow v\) the reverse edge \(v\rightarrow u\) is also present and moreover has same weight, then the problem of finding an optimal planar solution is called as \(\text{BI-DSN}_{\text{Planar}}\). For this problem, there is an \(1+\epsilon\)-approximation in \(\max\{ 2^{k^{2^{O(1/\epsilon)}}}, n^{2^{O(1/\epsilon)}} \}\) time. On the lower bound side, it is known that an \((1+\epsilon)\)-approximation is not possible in \(f(\epsilon,k)\cdot n^{o(\sqrt{k})}\) time (for any function \(f\)) under ETH [CFM18]
    .

  • Tree-cut width: Is the tree-cut width of a given undirected graph \(\leq w\)? See [W15] for a definition of tree-cut width.
    Algorithm: \(2\)-approximation in \(2^{O(w^{2}\cdot \log w)}\cdot n^{2}\) time [KOPST18]
    Lower Bound: None! It is known that there is a (non-constructive) FPT algorithm for tree-cut width parameterized by \(w\)

  • Steiner Orientation: Given a mixed graph \(G=(V,E)\) and a set of terminal pairs \(T=\{(s_i, t_i)\ :\ i\in [k]\}\) is there an orientation of the undirected edges of \(G\) such that the resulting graph has an \(s_i \leadsto t_i\) path for each \(i\in [k]\)
    Algorithm: None!
    Lower Bound: No \((19/20+\epsilon)\)-approximation (for any \(\epsilon>0\)) in \(f(k)\cdot n^{O(1)}\) time (for any function \(f\)) under Gap-ETH, even if the input graph is planar [CSF19]

  • \(k\)-Cut: Given an edge-weighted undirected graph \(G\) and an integer \(k\), can we remove a set of edges of weight \(\leq p\) so that G has at least \(k\) connected components.
    Algorithm: 1.81-approximation in \(2^{O(k^2)}\cdot n^{O(1)}\) time, and \((1+\epsilon)\)-approximation in \((k/\epsilon)^{O(k)}\cdot n^{k+O(1)}\) time for every \(\epsilon>0\) [GLL18]
    Lower Bound: None!

  • \(k\)-Median: Given a metric space \((V,d)\) with \(n\) points and clients at some of the points, the goal is to open \(k\) facilities such that the sum of distances from the clients to their closest facilities is minimized.
    Algorithm: For every \(\epsilon>0\), there is a \((1+\frac{2}{e}+\epsilon)\)-approximation algorithm that runs in \(f(k,\epsilon)\cdot n^{O(1)}\) time. [CGKLL19]
    Lower Bound: No \((1+\frac{2}{e}-\epsilon)\)-approximation (for any \(\epsilon>0\)) in \(f(k)\cdot n^{O(1)}\) time (for any function \(f\)) under Gap-ETH [CGKLL19]

  • \(k\)-Means: Given a metric space \((V,d)\) with \(n\) points and clients at some of the points, the goal is to open \(k\) facilities such that the sum of squares distances from the clients to their closest facilities is minimized.
    Algorithm: For every \(\epsilon>0\), there is a \((1+\frac{8}{e}+\epsilon)\)-approximation algorithm that runs in \(f(k,\epsilon)\cdot n^{O(1)}\) time. [CGKLL19]
    Lower Bound: No \((1+\frac{8}{e}-\epsilon)\)-approximation (for any \(\epsilon>0\)) in \(f(k)\cdot n^{O(1)}\) time (for any function \(f\)) under Gap-ETH [CGKLL19]
